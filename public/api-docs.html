üìã Available Endpoints

Recipes

GET /api/recipes - List all recipes
POST /api/recipes - Create a new recipe
GET /api/recipes/{id} - Get a specific recipe
PUT /api/recipes/{id} - Update a recipe
DELETE /api/recipes/{id} - Delete a recipe

Recipe Versions

GET /api/recipe_versions - List all recipe versions
POST /api/full_recipe_versions - Create a full recipe version
POST /api/link_recipe_versions - Create a link recipe version
POST /api/book_recipe_versions - Create a book recipe version

Ingredients & Steps

GET /api/ingredients - List all ingredients
POST /api/ingredients - Create a new ingredient
GET /api/steps - List all steps
POST /api/steps - Create a new step

üîß Recipe Version Types

  • Full Recipe Version: Complete recipes with ingredients, steps, and descriptions
  • Link Recipe Version: References to online recipes with URL and author
  • Book Recipe Version: References to recipes in books with title, author, and page number
